Abstract

The Some More Individual webspace index provides an overview of a couple of Semantic Web Ontologies and projects, which are related to them. They are mainly based in the media and personalisation domain, however, some of them are also representing basic modelling concepts, e.g. ordered lists. This document contains an RDFa description of the Some More Individual webspace index.

SMI Webspace Index

  • The Info Service Ontology
  • The Ordered List Ontology
  • The Counter Ontology
  • The Association Ontology
  • The Play Back Ontology
  • The Recommendation Ontology
  • The Weighting Ontology
  • The Cognitive Characteristics Ontology (and its predecessor the Weighted Interests Vocabulary)
  • The Property Reification Vocabulary
  • The My SpecGen version 6
